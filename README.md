# HydroSmiths Website (Fixed paths)

All assets use relative `./` paths so GitHub Pages loads CSS and images correctly.

## Upload steps
1. Upload *these files* to the **root** of your repo (not inside another folder):
   - `index.html`, `style.css`, `script.js`, and the `assets/` folder
2. In **Settings → Pages**, set Source to `main` and Folder `/root`.
3. Wait 1–2 minutes and refresh your site.
4. To verify CSS is live, open `/style.css` in your browser.
